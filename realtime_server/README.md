# 实时交通态势感知系统

基于 ASTGCN 深度学习模型的实时车辆速度预测系统。

## 快速启动

### 1. 启动后端服务

```bash
cd realtime_server
chmod +x start_backend.sh
./start_backend.sh
```

后端服务将在 `http://localhost:8000` 启动。

### 2. 启动前端服务

在新终端中运行：

```bash
cd realtime_server
chmod +x start_frontend.sh
./start_frontend.sh
```

前端服务将在 `http://localhost:3000` 启动。

### 3. 访问大屏

打开浏览器访问 `http://localhost:3000`

## 功能特性

### Tab 1: 宏观路网拓扑与预警监控

- **力导向网络图**: 基于 307 个节点的交通网络可视化
- **动态颜色编码**: 
  - 🟢 绿色: > 50 km/h (畅通)
  - 🟡 黄色: 20-50 km/h (缓行)
  - 🔴 红色: < 20 km/h (拥堵)
- **态势切换**: 一键切换"当前真实路况"与"未来1小时预测"
- **实时指标**: MAE/RMSE 实时性能仪表盘
- **拥堵预警**: TOP 5 拥堵节点滚动榜

### Tab 2: 微观时空推演与误差剖析

- **跑马灯折线图**: 向左无限滚动的动态时序图
- **三线对比**:
  - 青色实线: 过去3小时真实车速
  - 紫色虚线: 历史预测值
  - 橙色实线: 未来1小时预测
- **NOW 标记线**: 红色垂直线标记当前时刻
- **残差分析**: 底部误差柱状图，异常值红色预警
- **趋势判断**: 自动分析未来趋势向好/恶化

### Tab 3: 模型可解释性与时空溯源

- **注意力热力图**: 307×307 空间注意力矩阵可视化
- **节点关联分析**: 点击查看节点间的隐式依赖关系
- **迷你关系图**: 高亮显示选中节点对的连接
- **TOP 10 关联**: 最强空间依赖排行榜

## API 接口

### WebSocket

- `ws://localhost:8000/ws` - 实时数据推流

### REST API

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/topology` | GET | 获取网络拓扑结构 |
| `/api/node/{id}/history` | GET | 获取节点历史数据 |
| `/api/attention` | GET | 获取注意力矩阵 |
| `/api/attention/{id}` | GET | 获取节点关联 |
| `/api/congestion/top` | GET | 获取拥堵排行 |
| `/api/stats` | GET | 获取系统状态 |
| `/api/simulation/pause` | POST | 暂停模拟 |
| `/api/simulation/resume` | POST | 恢复模拟 |

## 数据流格式

WebSocket 每秒推送的 JSON 结构：

```json
{
  "timestamp": "2026-03-28 14:30:00",
  "system_metrics": {
    "current_mae": 1.89,
    "current_rmse": 3.91
  },
  "network_status": [
    {
      "node_id": 0,
      "current_real_speed": 65.2,
      "future_pred_speeds": [62.1, 58.0, 45.3, ...]
    }
  ]
}
```

## 技术栈

**后端:**
- FastAPI (异步 Web 框架)
- PyTorch (深度学习)
- WebSocket (实时通信)
- NumPy (数值计算)

**前端:**
- Vue 3 (Composition API)
- ECharts 5 (可视化)
- Element Plus (UI 组件)
- TailwindCSS (样式)
- Vite (构建工具)

## 配置说明

编辑 `backend/main.py` 中的 `CONFIG` 字典：

```python
CONFIG = {
    'num_of_vertices': 307,      # 节点数量
    'num_for_predict': 12,       # 预测步数
    'time_interval_minutes': 5,   # 时间间隔(分钟)
    'simulation_speed': 1.0,      # 模拟速度倍率
    'model_path': '...',          # 模型权重路径
    'data_path': '...',           # 数据文件路径
}
```
